Embedding Multimedia on a Web Page

<audio>Tag

HTML 5 has introduced a new audio element to embed a sound or audio file into HTML documents. There are various websites, such as YouTube, that facilitaies the user to play, listen or download the music files.Therefore, to play music on a Web page, you can embed music files that can be listened by the users.

Sometimes, a browser does not support the <audio> tag. In that case, you can display some content to the user, which informs the user that the browser does not support the tag. This content is placed between the opening and closing tag of <audio> and is termed as a fallback content. If the browser does not provide a support to the <audio> tag, fallback content is displayed on a Web page.

The element-specific attributes of the audio tag are listed :

AttributeValueDescription
autoplayautoplayPlays the sound file immediately or automatically on the loading of a Web Page
controlscontrolsDisplays the controls on a Web page, such as a play button or a stop button
preloadNone metadata autoStarts the buffering of the file
Src URLSpecifies the path of an audio file to be played
loop LoopIndicates to replay the audio file once it is finished

Example :

<!DOCTYPE html>
<html>
<head>
<title>Embedding Multimedia on a Web Page</title>
</head>
<body>
<audio controls="controls" loop="Loop" src="On my way.mp3">
</audio>
</body>
</html>




<Video>Tag

You can play a video file by using the <video> tag that has been introduced as a media element in HTML5. With the <video> tag, you can add the functionality to manipulate the videos and built-in video controls Web in a page. Along with this, you can also include a video player without installing a plug-in, such as Flash

The attributes of the <video> tag are listed :

AttributeValueDescription
autoplayautoplayPlays the video file immediately or automatically on the loading of a Web Page
controlscontrolsDisplays the controls on a Web page, such as a play button or a stop button
posterImage URLDisplays an image in case the video is unavailable or not loaded yet
preloadpreloadStarts the buffering of the file
Src URLSpecifies the path of an video file to be played
loop LoopIndicates to replay the video file once it is finished
height pixelsSpecifies the height for the video player displayed on a Web page
width pixelsSpecifies the width for the video player displayed on a Web page

Example :

<!DOCTYPE html>
<html>
<head>
<title>Embedding Multimedia on a Web Page</title>
</head>
<body>
<video controls="controls" height="500" width="300" src="Mee.mp4">
</video>
</body>
</html>




<source>Tag

HTML5 has introduced the <source> tag to embed two or more audio or video files of different media types.The browser may choose the type of file to play based on its media type or codec support. The <source> tag is used inside the media element, such as audio or video, to enmbed audio or video files. In other words, you can include more than one <source> tag within the media element.

The attributes of the <source> tag are listed :

AttributeValueDescription
Src URLSpecifies the path of the file name to play an audio or video file
type numeric valueIndicates the type of content that is embedded on a Web page
media Media querySpecifies the type of media resource, which is to be downloaded on the browser or not

Example :

<!DOCTYPE html>
<html>
<head>
<title>Embedding Multimedia on a Web Page</title>
</head>
<body>
<audio controls>
<source src="horse.ogg" type="audio/ogg">
<source src="On my way.mp3" type="audio/mpeg">
</audio>
</body>
</html>




<embed>Tag

There is another way to embed an audio file or a video file on a Web page by using the <embed> tag. Inside the <embed> tag, the src attribute is used to specify the path of the audio or video file you want to embed.

The attributes of the <embed> tag are listed :

AttributeValueDescription
Src URLSpecifies the path of the file name to play an audio or video file
type MIME typeIndicates the type of content that is embedded on a Web page
height pixelsSpecifies the height to display content on a Web page
widthpixelsSpecifies the width to display content on a Web page

Example :

<!DOCTYPE html>
<html>
<head>
<title>Embedding Multimedia on a Web Page</title>
</head>
<body>
<embed src="Astronaut.jpg" height="300" width="500">
</body>
</html>
