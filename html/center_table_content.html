Align and Cell Content of Table

You can provide the alignment to the table by using the align attribute in the <table> tag. A table contains cell that can also be aligned with the use of align attribute. Therefore, to apply the alignment to the cell conten you need to specify the align attribute with its value inside the <th> or <td> tag.

In this section, you learn to align a table in a Web page. In addition to this, you also learn to align the content a cell.

Example :

<!DOCTYPE html>
<html>
<head>
<title> Align and Cell Content of Table</title>
</head>
<body>
<table align="center" height="200" width="500" border="3">
<tr>
<th align="center">Person 1</th>
<th align="center">Person 2</th>
<th align="center">Person 3</th>
</tr>
<tr>
<td align="center">Emil</td>
<td align="center">Tobias</td>
<td align="center">Linus</td>
</tr>
<tr>
<td align="center">16</td>
<td align="center">14</td>
<td align="center">10</td>
</tr>
</table>
</body>
</html>
