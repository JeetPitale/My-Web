Adding Border to an Image

By default, an image is displayed without a border. However, a border can be added to an Image by using a border attribute of the <img> tag to specify the width of the border around an image.

Let's Adding a Border to an Image

Example :

<DOCTYPE html>
<html>
<head>
<title> Inserting an Image</title>
</head>
<body>
<img src="Space1.jpg" height=300" width="500" border="5"></img>
</body>
</html>
