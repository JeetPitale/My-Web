Creating an UNORDERED List

An unordered list is used to create a bulleted list of items. To display the list of items in the bulleted format, you can use the <ul> tag and the <li> tag. The <ul> tag is used to define an unordered list: whereas, the <li> tag is used to define the items of the list.

Let's create an unordered list:

Example :


<!DOCTYPE html>
<html>
<head>
<title>An Unordered List</title>
</head>
<body>
<h4>List of Fruits</h4>
<ul>
<li>Apple</1i>
<li>Banana</1i>
<li>Mango</li>
</ul>
<h4>List of vegetables</h4>
<ul>
<1i>0nion</1i>
<li>Capsicum</1i>
<li>Brinjal</1i>
</ul>
</body>
</html>


Types of Unordered List:

Default


<ul   type="1">
<li>Ice-Cream </li>
<li>Chocoba </li>
<li>Cold-Coffee </li>
</ul>

Square


<ul   style="list-style-type:square;">
<li>Sugar </li>
<li>Salt </li>
<li>Sour </li>
</ul>

Circle


<ul   style="list-style-type:circle;">
<li>Car </li>
<li>Bike </li>
<li>Scooty </li>
</ul>

None


<ul  style="list-style-type:none;">
<li>Airplane </li>
<li>Bus </li>
<li>Ship </li>
</ul>
